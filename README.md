# Registro CES Montessori

Sistema de gestión académica y control de estudiantes para CES Montessori.

## Estructura del proyecto
- `index.html`: Portal de acceso y autenticación.
- `app.html`: Panel administrativo para gestión de expedientes y calificaciones.
- `CES MONTESORI.sql`: Respaldo del esquema de la base de datos.

## Tecnologías
- Frontend: Tailwind CSS
- Backend: Supabase
